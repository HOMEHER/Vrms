方均根速率 (Root Mean Square Speed)

什麼是方均根速率？

方均根速率是描述氣體分子運動的重要物理量之一。它代表了氣體分子速率的統計平均值，考慮了所有分子的速率分布。

方均根速率的計算公式為：

\[v_{rms} = \sqrt{\frac{3RT}{M}} = \sqrt{\frac{3k_BT}{m}}\]

其中：

  • \(v_{rms}\) 是方均根速率
  • \(R\) 是氣體常數 (8.314 J/(mol·K))
  • \(T\) 是絕對溫度 (K)
  • \(M\) 是氣體的摩爾質量 (kg/mol)
  • \(k_B\) 是波茲曼常數
  • \(m\) 是分子質量

高中物理例題

基礎例題 1：

在室溫（25°C）下，氮氣（N₂）分子的方均根速率為多少？（氮氣的摩爾質量為 28 g/mol）

解答：

已知條件：

  • 溫度 T = 25°C = 298 K
  • 摩爾質量 M = 28 g/mol = 0.028 kg/mol
  • 氣體常數 R = 8.314 J/(mol·K)

代入公式 \(v_{rms} = \sqrt{\frac{3RT}{M}}\)：

\(v_{rms} = \sqrt{\frac{3 \times 8.314 \times 298}{0.028}} = 517\) m/s

基礎例題 2：

某氣體在 300K 時的方均根速率為 400 m/s，請問這個氣體的摩爾質量為多少？

解答：

使用公式 \(v_{rms} = \sqrt{\frac{3RT}{M}}\) 反推：

\(400 = \sqrt{\frac{3 \times 8.314 \times 300}{M}}\)

\(M = \frac{3 \times 8.314 \times 300}{400^2} = 0.047\) kg/mol = 47 g/mol

進階例題：

在相同溫度下，氫氣(H₂, M = 2 g/mol)和氧氣(O₂, M = 32 g/mol)的方均根速率比為何？

解答：

對於兩種氣體，溫度T和氣體常數R都相同，因此：

\(\frac{v_{rms(H_2)}}{v_{rms(O_2)}} = \sqrt{\frac{M_{O_2}}{M_{H_2}}} = \sqrt{\frac{32}{2}} = 4\)

這說明在相同溫度下，氫氣分子的方均根速率是氧氣的4倍。這也解釋了為什麼較輕的氣體分子逸散速率較快。

從氣體動力論到方均根速率

1. 氣體壓力的推導

讓我們從最基本的觀念開始：氣體分子不斷地撞擊容器壁，這些撞擊產生了我們感受到的氣體壓力。

推導步驟：

  1. 想像一個邊長為 L 的正立方體容器，裡面有一個氣體分子：

    • 分子質量：m
    • x 方向速度：vx

  2. 分子撞擊容器壁時：

    • 動量變化：Δp = 2mvx（來回碰撞）
    • 碰撞時間間隔：Δt = 2L/vx（來回一次的時間）

  3. 根據牛頓第二定律，力等於動量變化率：

    \[F = \frac{\Delta p}{\Delta t} = \frac{2mv_x}{2L/v_x} = \frac{mv_x^2}{L}\]

  4. 壓力是力除以面積：

    \[P = \frac{F}{A} = \frac{mv_x^2}{L^3} = \frac{mv_x^2}{V}\]

  5. 考慮所有方向和 N 個分子：

    \[P = \frac{Nm\overline{v^2}}{3V}\]

    其中 \(\overline{v^2}\) 是速率平方的平均值

2. 分子的平均動能與溫度

接下來，我們來看看分子運動與溫度的關係。溫度其實就是分子平均動能的量度！

關鍵概念：

  1. 一個分子的動能：

    \[E_k = \frac{1}{2}mv^2\]

  2. 實驗發現：分子的平均動能正比於絕對溫度

    \[\overline{E_k} = \frac{1}{2}m\overline{v^2} = \frac{3}{2}k_BT\]

    其中 kB 是波茲曼常數，T 是絕對溫度

  3. 將這個關係代入壓力公式：

    \[P = \frac{N}{V}k_BT\]

    這就是理想氣體方程式！

3. 單原子理想氣體的內能

現在我們來看看氣體的總能量（內能）。

推導過程：

  1. 對於單原子氣體（如氦氣）：

    • 只有平動動能，沒有轉動和振動能
    • 每個分子的平均動能是 \(\frac{3}{2}k_BT\)

  2. 總內能（N 個分子）：

    \[U = N\overline{E_k} = N \cdot \frac{3}{2}k_BT = \frac{3}{2}nRT\]

    其中 n 是氣體的摩爾數，R 是氣體常數

4. 方均根速率

最後，我們來看看分子的平均速度。方均根速率是一種特別有用的平均速度。

從前面的推導得到：

  1. 我們知道：\(\frac{1}{2}m\overline{v^2} = \frac{3}{2}k_BT\)

  2. 整理得到：\(\overline{v^2} = \frac{3k_BT}{m}\)

  3. 方均根速率就是這個值的平方根：

    \[v_{rms} = \sqrt{\overline{v^2}} = \sqrt{\frac{3k_BT}{m}} = \sqrt{\frac{3RT}{M}}\]

物理意義：

  • 方均根速率代表氣體分子運動快慢的一種平均值
  • 溫度越高，分子運動越快
  • 分子質量越大，在相同溫度下運動越慢
  • 這就解釋了為什麼氫氣會比氧氣擴散得快！

單原子理想氣體的巨觀與微觀比較

巨觀性質

性質 描述
壓力 (P) 氣體對容器壁的作用力與面積之比
體積 (V) 氣體佔據的空間大小
溫度 (T) 反映氣體的熱力學狀態
摩爾數 (n) 氣體的物質量

微觀性質

性質 描述
分子運動 分子做無規則熱運動（布朗運動）
分子速率 符合麥克斯韋-波茲曼分布
分子間碰撞 完全彈性碰撞，動能守恆
分子平均動能 \(\overline{E_k} = \frac{3}{2}k_BT\)

巨觀與微觀的關聯

  1. 壓力與分子運動：
    • 巨觀：容器內的壓力均勻分布
    • 微觀：來自於大量分子撞擊容器壁的統計平均效應
    • 關係式：\(P = \frac{1}{3} \cdot \frac{N}{V}m\overline{v^2}\)
  2. 溫度與分子動能：
    • 巨觀：溫度是熱力學平衡狀態的表徵
    • 微觀：反映了分子平均動能的大小
    • 關係式：\(T \propto \overline{E_k}\)
  3. 方均根速率的意義：
    • 巨觀：氣體擴散和混合的速率指標
    • 微觀：分子運動速率的統計平均值
    • 關係式：\(v_{rms} = \sqrt{\frac{3RT}{M}} = \sqrt{\frac{3k_BT}{m}}\)

重要結論

  • 巨觀性質是大量分子微觀行為的統計結果
  • 溫度越高，分子運動越劇烈，方均根速率越大
  • 在相同溫度下，質量較小的分子方均根速率較大
  • 理想氣體方程式 PV = nRT 是巨觀與微觀性質的完美橋樑
